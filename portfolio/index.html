WEEBOBOT is a twitch chat moderation bot written in Java using PIrcBot. I am no longer actively working on this project. The project is approximately 60% complete with majority of the functionality implementation left to be done on the Web frontend. There is also a sister-project DWeeboBot which is the Discord variation on the bot itself. This project uses Discord4J to communicate with the Discord API and was started in September of 2016. It is still in the early stages of development.

DISCORDORAGI is a bot for Discord written in Python using discord.py. The bot is a fork of a Reddit bot called Roboragi which fetches Anime, Manga, and Light Novels from various databases and returns a formatted answer with various information. This project has been taken over by Dashwav.

LiveBroadcast is an announcement plugin that I built early on in my Java programming days for Craftbukkit servers. The plugin went through several major rewrites as I learned better practices for writing more efficient code. It stands today as the closest I have ever come to "finishing" a project.

CYOA is a Choose Your Own Adventure game built in Java using Swing it is a project that I started in High School with a class mate and continue to work on little by little as time/inspiration allows.
