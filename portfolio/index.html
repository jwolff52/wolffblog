WEEBOBOT is a twitch chat moderation bot written in Java using PIrcBot. It is currently in development at this time. The project is approximately 60% complete with majority of the functionality implementation left to be done on the Web frontend. There is also a sister-project DWeeboBot which is the Discord variation on the bot itself. This project uses Discord4J to communicate with the Discord API and was just started in September of 2016. It is still in the early stages of development with the first "working version" planned to be ready by December of 2016.

DISCORDORAGI is a bot for Discord written in Python using discord.py. The bot is a fork of a Reddit bot called Roboragi which fetches Anime, Manga, and Light Novels from various databases and returns a formatted answer with various information. An example of an anime request from both bots can be found here and here. One of the largest issues is that Discord does not support links in their Markdown implementation which causes responses to be considerably longer and not near as pretty as their Reddit counterparts. If you would like more information visit the site!
